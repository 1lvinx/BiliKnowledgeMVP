# Manual UI Acceptance Checklist

## Overview Page

- [ ] Mouse wheel scrolls normally
- [ ] Bottom content can be reached
- [ ] No horizontal scrollbar
- [ ] Sidebar stays fixed
- [ ] Header layout is stable
- [ ] No useless browser-like back/forward buttons
- [ ] No meaningless List/Detail toggle
- [ ] Focus cards have consistent spacing
- [ ] Focus cards have consistent radius
- [ ] Focus cards have consistent shadow
- [ ] Metric cards are visually lighter than focus cards
- [ ] Hero area is not visually overwhelming
- [ ] Quick actions are clearly clickable
- [ ] No visible untranslated status labels
- [ ] Page still works at 1440px width
- [ ] Page still works at 1100px width

## Favorites Page

- [ ] List and right detail panel align correctly
- [ ] Selected row state is visible
- [ ] Detail panel does not visually crush the list
- [ ] Action buttons have real handlers or disabled state
- [ ] No large broken blank area
- [ ] Page can scroll when content exceeds viewport

## Video Page

- [ ] Analyze button is clear
- [ ] Right detail panel does not visually crush the list
- [ ] Filters align cleanly
- [ ] Selected item state is visible
- [ ] No fake clickable controls
- [ ] Page can scroll when content exceeds viewport

## Projects Page

- [ ] Left project list is readable
- [ ] Middle detail area has enough width
- [ ] Right review panel does not over-compress the content
- [ ] Narrow windows automatically degrade layout
- [ ] Status labels are translated or intentionally technical

## Notes Page

- [ ] Note list is readable
- [ ] Reading panel is comfortable
- [ ] Empty state is clear
- [ ] Long note content scrolls correctly

## Settings Page

- [ ] Sidebar and settings content do not overlap
- [ ] Tabs are not floating over content awkwardly
- [ ] Form labels are readable
- [ ] Save button is visible and has clear behavior
- [ ] Page scroll works
- [ ] No untranslated visible labels

## Global

- [ ] No horizontal scrollbar
- [ ] No click-without-feedback buttons
- [ ] No broken overflow clipping
- [ ] No large visual jumps between pages
- [ ] Primary action buttons are visually consistent
- [ ] Secondary action buttons are visually consistent
- [ ] Text contrast is readable
- [ ] Spacing is consistent
