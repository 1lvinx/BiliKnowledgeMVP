## 3. 核心内容摘要

### 3.1 主要观点

- 2026年 AI 开源生态已经从「模型竞争」转向「工具链竞争」
- Agent 框架是最热门方向，AutoGPT、CrewAI、LangGraph 三足鼎立
- 本地推理能力大幅提升，Ollama + llama.cpp 成为标配

### 3.2 关键步骤 / 方法

1. Agent 框架选型（LangGraph vs CrewAI vs AutoGen）
2. 本地模型部署（Ollama + GGUF）
3. 数据处理管道（Unstructured + LlamaIndex）

### 3.3 重要概念

| 概念 | 解释 | 价值 |
|---|---|---|
| Agent框架 | 自主决策+工具调用的AI系统 | 核心趋势 |
| 本地推理 | 不依赖云API的模型运行方式 | 降低成本 |
