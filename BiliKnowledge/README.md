# BiliKnowledge

这是一个用于整理 B 站收藏夹、视频笔记、开源项目和个人想法的本地 Markdown 知识库。

## 目录说明

| 目录 | 用途 |
|---|---|
| manifest/ | 收藏夹视频清单 |
| notes/raw/ | 原始视频笔记 |
| notes/reviewed/ | 人工复核后的正式笔记 |
| projects/ | 视频中提到的开源项目、工具、框架 |
| thoughts/ | 个人想法、产品灵感、商业化判断 |
| scripts/ | 数据处理脚本 |
| reports/ | 执行报告与错误日志 |

## 工作流

1. 导出 B 站收藏夹；
2. 生成视频清单；
3. 挑选高价值视频；
4. 生成视频笔记；
5. 提取项目和想法；
6. 生成索引；
7. 人工复核归档。
