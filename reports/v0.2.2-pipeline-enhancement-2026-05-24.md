## Mission

Enhance the BiliKnowledgeMVP pipeline status card with three new features:
1. **Run Full Pipeline** — one-click execution of all pipeline scripts in sequence
2. **Pipeline Step Progress** — real-time indicator showing which script is running
3. **Timestamp-based Staleness Detection** — warns when data is older than 24 hours

## Changes Made

### Frontend (App.tsx)

1. **New state variables**
   - `pipelineRunning: boolean` — tracks full pipeline execution
   - `pipelineStep: string | null` — tracks current script name

2. **New functions**
   - `runFullPipeline()` — runs all 4 scripts in sequence: `parse_favorites.py` → `extract_projects.py` → `build_index.py` → `validate_knowledge_base.py`
   - `getStaleness(lastUpdated)` — calculates hours since last update, flags >24h as stale

3. **UI updates**
   - Added "Run Full Pipeline" button in pipeline status card header
   - Added pipeline step progress indicator (shows current script name)
   - Updated staleness notice to show hours since last update
