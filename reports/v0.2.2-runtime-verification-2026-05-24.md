### 2.3 Staleness Detection

**Code location:** `App.tsx:715-720, 1015-1034`

```typescript
function getStaleness(lastUpdated: string | undefined): { stale: boolean; hours: number } {
  if (!lastUpdated) return { stale: false, hours: 0 };
  const updated = new Date(lastUpdated);
  const now = new Date();
  const hours = (now.getTime() - updated.getTime()) / (1000 * 60 * 60);
  return { stale: hours > 24, hours: Math.round(hours) };
}
```

**Verification checklist:**
- [x] Function defined outside component (pure utility)
- [x] Calculates hours since last update
- [x] Returns stale=true when >24 hours
- [x] UI shows warning with hours count
- [x] Falls back to "flags not complete" notice if not stale but flags incomplete

**Expected behavior:**
1. If `last_updated` is >24 hours ago, shows: "Data is {hours}h old. Consider running the pipeline to refresh."
2. If `last_updated` is <24 hours but flags incomplete, shows: "Pipeline status may be stale. Some flags are not complete."
3. If all flags true and data fresh, no notice shown
