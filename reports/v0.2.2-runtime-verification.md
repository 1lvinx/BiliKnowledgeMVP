## 3. 24-Hour Staleness Detection

**Code Location:** `src/App.tsx:715-721,1017-1024`

**Implementation:**
```typescript
function getStaleness(lastUpdated: string | undefined): { stale: boolean; hours: number } {
  if (!lastUpdated) return { stale: false, hours: 0 };
  const updated = new Date(lastUpdated);
  const now = new Date();
  const hours = (now.getTime() - updated.getTime()) / (1000 * 60 * 60);
  return { stale: hours > 24, hours: Math.round(hours) };
}
```

**UI Display:** Lines 1018-1024
```tsx
const { stale, hours } = getStaleness(pipelineStatus.last_updated);
if (stale) {
  return (
    <MacInlineNotice tone="neutral">
      <Circle size={10} fill="currentColor" /> Data is {hours}h old. Consider running the pipeline to refresh.
    </MacInlineNotice>
  );
}
```

**Verification Result:** ✅ PASS

- Calculation: `(now - lastUpdated) / (1000 * 60 * 60)` gives hours
- Threshold: `hours > 24` triggers staleness warning
- UI shows: "Data is {hours}h old. Consider running the pipeline to refresh."
- Fallback: If pipeline flags incomplete, shows alternative message
